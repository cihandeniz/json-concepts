# Separators

```javascript
module.exports = { /* # */$function };
```

```javascript
module.exports = { /* # */$function/* ,/ */ };
```

```javascript
module.exports = { /* #function */$function/* ,/ */ };
```

```javascript
module.exports = { /* #function */$function/* , *//* / */ };
```

```javascript
module.exports = { /* #function */$function/* !_last */,/* / *//* / */ };
```
